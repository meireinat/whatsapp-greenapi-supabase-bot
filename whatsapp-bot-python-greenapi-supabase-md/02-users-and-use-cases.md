# משתמשי יעד ותסריטי שימוש

## משתמשי יעד

- מנהלי משמרת, ראשי מדורים ומנהלי מחלקות.
- בקרי תפעול, מתכננים.
- עובדים במטה הזקוקים למידע תפעולי מהיר ב-WhatsApp.

## תסריטי שימוש עיקריים

### 1. סטטיסטיקות יומיות

- "כמה מכולות פרקו היום?"
- "כמה רכבים עברו ברמפה 3 במשמרת בוקר?"
- "כמה אוניות פעילות כרגע?"

### 2. שאילתות היסטוריות

- "כמה מכולות עברו בשבוע האחרון דרך רמפה 2?"
- "מה היה מספר הרכבים הממוצע ליום בחודש האחרון?"
- "כמה תקלות נרשמו במערכת X אתמול?"

### 3. שאילתות על אובייקט ספציפי

- "איפה נמצאת מכולה ABCD1234567?"
- "מה הסטטוס של אוניה MSC XXX?"
- "כמה פעמים מכולה זו זזה בחודש האחרון?"

### 4. שאילתות ניהוליות

- "מי עבד במשמרת בוקר ברמפה 2 היום?"
- "כמה משמרות לא היו מאוישות במלואן השבוע?"
- "כמה שעות עבודה נרשמו במדור Y אתמול?"

כל תסריט שימוש יתורגם ל-Intent מוגדר יחד עם פרמטרים (תאריך, מזהה מכולה, מזהה אוניה, משמרת, רמפה וכו'), עליהם יישב מנגנון ה-NLU/LLM.
