# מטרת הפרויקט

להקים בוט שאילתות ארגוני הפועל מעל WhatsApp, המתחבר דרך Green API אל שירות Python, ומשתמש ב-Supabase (Postgres מנוהל בענן) כמקור נתונים מרכזי.

הבוט יאפשר לעובדי הארגון לשאול בשפה טבעית (בעברית/אנגלית) שאלות תפעוליות וסטטיסטיות (כמו כמות מכולות, סטטוס אוניות, תפוקות לפי משמרת וכו'), ולקבל תשובות בזמן אמת, מבוססות על נתונים הנשמרים ב-Supabase.

המערכת תתוכנן כך שתהיה:
- מאובטחת (גישה מוגבלת, ניהול מפתחות, RLS ב-Supabase).
- ניתנת להרחבה (הוספת Intents חדשים, הרחבת טבלאות).
- קלה לתחזוקה (קוד Python מודולרי, לוגים מסודרים).
